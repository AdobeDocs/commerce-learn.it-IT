# SQL REPORT BUILDER

Adobe Commerce Intelligence SQL Report Builder consente di eseguire query nel data warehouse, esaminare i risultati e trasformarli in grafici. Quando si creano report con SQL, non si attendono cicli di aggiornamento per perfezionare le colonne create. Se qualcosa non funziona, modifica ed esegui nuovamente la query fino a quando l’output non corrisponde a quello previsto.

>[!IMPORTANT]
>
> Per utilizzare SQL Report Builder sono necessarie le autorizzazioni di amministratore.

>[!TIP]
>
> Se sei un nuovo utente di Commerce Intelligence, la [guida utente di Adobe Commerce Intelligence](https://experienceleague.adobe.com/docs/commerce-business-intelligence/mbi/guide-overview.html?lang=it){target="_blank"} spiega come aggregare, analizzare e visualizzare i dati per ottenere informazioni aziendali più approfondite.
