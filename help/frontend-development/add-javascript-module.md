## Passaggi

- Creare un modulo
- Creazione di un file di modulo `requirejs-config.js` e JavaScript
- Crea aggiornamento layout per aggiungere un modello che abilita il modulo JS
- Crea file modello
- Aggiungi il modulo e testalo
